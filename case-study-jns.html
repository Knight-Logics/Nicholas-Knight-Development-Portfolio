The Build

Hand-Coded. Fully Custom. Zero Templates.

HTML5 CSS3 Vanilla JS GitHub Pages Formsubmit.co WebP Images JSON-LD Schema Google Search Console

JNS Construction needed a professional online presence to back up their licensed contractor credentials and compete in local search for Palm Harbor and Pinellas County. No page builder, no WordPress — just clean, hand-coded HTML deployed on GitHub Pages.

The site is structured with 11 pages: homepage, about, contact, four service pages (General Construction, Renovations & Upgrades, Repairs & Corrective Work, Project Coordination), and four policy pages. Every page carries its own SEO metadata, Open Graph tags, Twitter Cards, and canonical links.

Performance was treated as a first-class priority: compressed WebP hero images with `fetchpriority="high"` preloading, no render-blocking resources, and zero layout shift on load — resulting in a 0.7s LCP on desktop.

What Was Delivered

  • 11-page static site with full responsive design (mobile, tablet, desktop)
  • GeneralContractor + LocalBusiness JSON-LD schema with license number (CRC1334879)
  • FAQPage schema with accordion UI on all 4 service pages
  • BreadcrumbList schema on all inner pages
  • Open Graph + Twitter Card meta on every page
  • Sitemap.xml submitted and verified in Google Search Console
  • robots.txt configured, canonical tags on all pages
  • Contact form via Formsubmit.co routed to business email
  • Custom favicon (512px branded icon, dark-bg) for Google Search visibility
  • 100/100 Accessibility score — heading order, contrast ratios, ARIA labels
