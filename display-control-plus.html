Display Control+

Windows utility for OLED/LCD burn-in protection and multi-monitor idle overlays.

Built by Knight Logics for users who need reliable monitor protection with simple controls and persistent settings.

Best File for Most Users

  • tray.exe - recommended startup app with dashboard access and pause/resume controls

Quick Start

  1. Download and run tray.exe.
  2. Right-click the tray icon and open the dashboard.
  3. Choose displays, timeout, and overlay mode.
  4. Click Apply.

Core Features

  • Per-monitor idle detection and protection profiles
  • Overlay modes: blank, single image, slideshow, video playlist
  • Tray controls: open dashboard, pause/resume, exit
  • Settings persist through reboots
  • Auto-update prompt when a newer release is published
