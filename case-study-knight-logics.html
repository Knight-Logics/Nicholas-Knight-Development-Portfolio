The Build

A Portfolio That Practices What It Preaches

HTML5 CSS3 Vanilla JS Vercel GA4 Analytics JSON-LD Schema Google Search Console WebP Images

If the goal is to sell website development, the website that sells it has to be the best example of it. KnightLogics.com was built to that standard — hand-coded, performance-first, and fully instrumented with analytics and structured data from the start.

The site has multiple service pages, a project showcase system, contact forms, policy pages, and a structured content hierarchy that Google can crawl and understand. The header and footer load dynamically via JavaScript fetch, keeping the codebase clean without a backend or build system.

Every page carries full SEO metadata, Open Graph tags, Twitter Cards, canonical links, and JSON-LD schema. Google Analytics 4 is integrated for conversion tracking, and the site is verified and monitored in Google Search Console.

What Was Built

  • Multi-page portfolio: home, projects, 4 service pages, contact, policy pages
  • Dynamic header/footer loading via JS fetch — DRY architecture, no build step
  • Project showcase with demo videos and live links
  • Person + Organization JSON-LD schema for entity establishment
  • Service schema on each service page with areaServed
  • BreadcrumbList on all inner pages
  • Open Graph + Twitter Card meta on every page
  • Google Analytics 4 with custom config
  • Sitemap.xml submitted to Google Search Console
  • 100/100 Accessibility — full WCAG compliance
  • Hosted on Vercel with edge CDN and automatic HTTPS
